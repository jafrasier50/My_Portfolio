Lofty Pursuits Website

Web Design

Just a redesign of the Lofty Pursuits Website. It's made in React, and all of the CSS is custom. However, The site itself is not the main attraction.

Lofty Pursuits Candy Machine

Web Development

The main attraction of the Lofty Pursuits Website was actually recycled from the Market of the Seas project. The Candy Machine allows the customer to manage and order their candy online as well as see their totals before checking out all in one space. It also allows the authenticated owner to edit inventory in real time. This project was made in React, uses Stylus powered CSS, and it's all tied into a Firebase Database with Facebook Authentification.

Houston's Pet Service Hub.

Web Design

As a Group Front-End project, Houston's Pet Service hub was the idea to take a pet service and make it a little more personal. Users make an account, sign in, add thier pets, and then make a post about what ever simple service needed to the community around them. This app features Javascript, some CSS(I'm responsible), Bootstrap(I'm responsible), and hooked into a Firebase database.

Market of the Seas

Web Development

An inventory management system that allows users to add up thier order while allowing an authenticated owner to sign in through Facebook or Github. This project is made in React, uses Stylus powered CSS, and is connected to a Firebase database Try The Demo Here: Market of the Seas

LIBRARYMGMT

Web Design

A group project that was a simple app that served as a library inventory system where staff can add, remove, or edit book information, while keeping track of who was using the book. It also gives ordinary users who are looking for books to check any library using the app to see if their book was in or not, and if it was they could reserve the book for pick up. This app was made using Mustache, and SQL. The UI (I'm responsible) was made with some CSS and Bootstrap.
